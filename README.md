# MirrorStudy MultiPassRendering

DirectX 9 のマルチパスレンダリングで、鏡面反射を描画するサンプルです。

## 概要

このプログラムは、通常の描画結果とは別に鏡用のレンダーターゲットを作成し、鏡カメラから見たシーンを先に描画します。その反射用テクスチャを通常シーンの鏡面へ投影することで、鏡に周囲のオブジェクトが映り込むように見せています。

## ファイル

- `main.cpp`: Direct3D の初期化、カメラ操作、メッシュ読み込み、レンダーパス制御を行います。
- `simple.fx`: 3D メッシュを描画するエフェクトです。通常描画と鏡面への反射テクスチャ投影を扱います。
- `simple2.fx`: 中間レンダーターゲットを画面へ表示するエフェクトです。
- `cube.x`: サンプル用のメッシュです。
- `grass.png`: サンプル用のテクスチャです。

## 操作方法

- `W` / `A` / `S` / `D`: カメラを前後左右へ移動します。
- `Q` / `E`: カメラを上下へ移動します。
- マウス移動: 視点を回転します。
- `Esc`: カーソルロックを切り替えます。
- `F1`: X ファイルを読み込み、カメラ前方へ配置します。

## 描画の流れ

1. 鏡面メッシュから鏡平面を求めます。
2. 通常カメラを鏡平面で反転し、鏡カメラを作成します。
3. 鏡カメラでシーンを描画し、鏡用レンダーターゲットへ保存します。
4. 通常カメラでシーンを描画し、鏡面だけ反射用テクスチャを参照します。
5. 最後に中間レンダーターゲットをバックバッファへ転送します。
